完成 Zent v4 核心架构实现：
- ✅ 强类型 MemoryStep 系统
- ✅ Agent 抽象基类 (Template Method)
- ✅ ToolCallingAgent 实现
- ✅ ToolRegistry 工具注册表
- ✅ @tool 装饰器
- ✅ create_agent 工厂函数
- ✅ 基础集成 (OpenAI, InMemoryMemory)

## 6. 风险与应对

| 风险 | 可能性 | 影响 | 应对 |
|:---|:---|:---|:---|
| OpenAI API 测试依赖 | 中 | 高 | 使用 Mock，避免真实调用 |
| Async 复杂度 | 低 | 中 | 保持简单 async/await 模式 |
| 类型兼容问题 | 中 | 中 | 使用 Protocol + runtime_checkable |
